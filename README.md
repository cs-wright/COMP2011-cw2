# COMP2011-cw2
A simple social media web application

To run this web application you will need to setup a python virtual environment by following the instructions below:

First create a virtual environment
```$ python3 -m venv flask``` note flask can be substituted for any name

Then clone this repo.

Activate the virtual environment ```$source /path/to/virtual/env/flask/bin/activate```

Next, install pip install the modules required to run the application these are provided in the repo.
```$pip install -r /path/to/virtual/repo/COMP2011-cw2/requirements.txt```

Now the environment is configured, so we just need to launch the web application. To do this we run ```$flask run```

The website is now available to you on your local machine, open a web browser an navigate to ```http://127.0.0.1:5000/```

To access the deployed version of the website enter this url into your browser https://sc19c2sw.pythonanywhere.com/
